Administrador

L'aplicació administrador només accessible per a l'administrador del sistema i està dissenyada per configurar l'entorn. Amb aquesta aplicació podrà:

  • Configurar les connexions a Mysql
  • Canviar contrasenyes
  • Canviar d'any acadèmic o crear un nou curs
  • Importar dades des SGD
    • Definir llistes d'importació d'incidències
  • Editar usuaris
    • Crear un nou usuari
  • Editar alumnes
  • Editar dates d'avaluacions i festius
  • Definir regles per a la gestió d'actuacions (Rules)
    • Creació del formulari en xml
    • Associar i compilar informes Jasper
  • Definir grups d'usuaris i gestionar permisos (Roles)
  • Editar fitxers JavaScript per al java scriptEngine

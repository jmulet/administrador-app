Administrator

The application administrador is only accessible for the system administrator and is designed to set up the environment. With this application you can:

  • Configure Mysql connections 
  • Change Passwords
  • Change academic year or create a new course
  • Import data from SGD
    • Define import lists of incidences
  • Edit users
    • Create a new user
  • Edit students
  • Edit terms dates and holidays
  • Define rules for the management of actions
    • Create a form in xml
    • Associate and compile Jasper reports
  • Define user groups and manage permissions (roles)
  • Edit JavaScript files for java ScriptEngine

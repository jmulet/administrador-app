Administrador

La aplicación administrador solo es accesible para el administrador del sistema y esta diseñada para configurar el entorno. Con esta aplicación podrá:

  • Configurar las conexiones a Mysql
  • Cambiar contraseñas
  • Cambiar de año académico o crear un nuevo curso
  • Importar datos desde SGD
    •  Definir listas de importación de incidencias
  • Editar usuarios
    • Crear un nuevo usuario
  • Editar alumnos
  • Editar fechas de evaluaciones y festivos
  • Definir reglas para la gestión de  actuaciones (Rules)
    • Creación del formulario en xml
    •  Asociar y compilar reportes Jasper
  • Definir grupos de usuarios y manejar permisos (Roles)
  • Editar ficheros JavaScript para el java scriptEngine
